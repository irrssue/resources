Must-Read Books

Clean Code

Book

By Robert C. Martin. A handbook of agile software craftsmanship that has helped countless programmers write better, more maintainable code.

The Pragmatic Programmer

Book

By Andrew Hunt and David Thomas. Cutting-edge software development practices that help you build better software.

Design Patterns

Book

By Erich Gamma, Richard Helm, Ralph Johnson, John Vlissides. Classic book on reusable design patterns in object-oriented software.
